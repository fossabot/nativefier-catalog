# nativefier-catalog

> Calalog of nativefier apps

## Install

```
$ npm install nativefier-catalog
```

## Usage

```js
import nativefierCatalog from 'nativefier-catalog';

nativefierCatalog('unicorns');
//=> 'unicorns & rainbows'
```
